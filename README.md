Agenda Nails

*Status do Projeto*: 
Em desenvolvimento

Descrição: 
Agenda Nails é um sistema desktop desenvolvido para gerenciar agendamentos de horários em serviços de manicure e pedicure. O sistema permite o cadastro de profissionais, cadastro de clientes e o agendamento de horários conforme a disponibilidade de cada profissional.

Tecnologias utilizadas:
- Java
- Banco de dados (MySQL)
- Git/ GitHub (para versinamento de código)

  Objetivo:
  O objetivo do sistema é facilitar a organização de atendimentos nas esmalterias, evitando conflitos de agenda e otimizando o tempo de trabalho das profissionais.

  Funcionalidades:
  - Cadastro de profissionais (manicures e pedicures)
  - Cadastro de clientes
  - Agendamento de horários
  - Consulta de agenda por profissional
  - Edição e cancelamento de agendamentos
